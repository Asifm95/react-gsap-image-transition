## Install

    $ git clone https://github.com/Asifm95/react-gsap-image-transition.git
    $ cd react-gsap-image-transition
    $ npm install

## Start & watch

    $ npm start

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Simple build for production

    $ npm run build
